# RhombixTechnologies_Task4
Dev Engima is a modern, high-performance web platform designed for software engineers, UI/UX designers, and systems architects to publish, read, and manage technical articles and system design breakdowns. Built with vanilla JavaScript, modern CSS custom properties, and semantic HTML5, DevPulse offers a seamless, single-page application (SPA) experience featuring a theme toggle, rich content editing, bookmarking, and administrative controls.

🌟 Key Features
🎨 Modern & Responsive Design
Light & Dark Theme Engine: Smooth dynamic switching using CSS variable architecture and dataset state persistence (☀️ Light / 🌙 Dark).

Glassmorphism & Depth Elements: Backdrop blur filters, elevated card shadows, glowing accent borders, and tailored typography.

Responsive Layout: Adaptive CSS grid and flexbox structure supporting desktop, tablet, and mobile displays.

📰 Interactive Article Reader & Feed
Unsplash Banner Integration: High-resolution cover images with visual fallback options mapped to specific technical categories.

Instant Search & Category Filtering: Real-time search across article titles, tags, and excerpts alongside dynamic category selection (Engineering, Design, Architecture, Telecom NOC, Career).

Bookmark & Saved Articles System: Quick-filter toggle to save and access bookmarked articles instantly during a session.

Like & Social Sharing: Interactive like counters with real-time updates and one-click clipboard link sharing.

Interactive Comments Engine: Article discussion board allowing readers to share feedback and ask technical questions.

✍️ Article Creator & Rich Text Editor
WYSIWYG Formatting: Full support for rich text controls, including Headings (H2, H3), lists, blockquotes, bold, italic, and underline styling.

Image Upload & Live Preview: Local image file uploading via FileReader with instant preview and option to reset.

Custom Tagging System: Input comma-separated tags rendered into interactive badge clouds.

🛡️ Administrative Portal
Passcode Protection: Secure client-side administrative dashboard unlocked via passcode (Default: admin123).

Real-time System Metrics: Track total articles published, active user comments, and active categories.

Data Management Table: Live administration table to inspect engagement metrics and delete entries directly from state.
